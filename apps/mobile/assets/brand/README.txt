UniSync — Brand Assets
======================

The mark: "Quad" — an abstract campus hall that assembles itself on load.
Foundation = belonging · Columns = community (one ochre = the individual)
Roof = aspiration · Emblem = you, the spark.

Palette
-------
Teal (primary)        #2E5559
Teal (deep / shadow)  #1F3D40
Ochre (accent)        #B5831E
Ochre (light)         #E7B85C
Cream (surface)       #FAF8F5
Ink (text)            #28241D

Typeface: Plus Jakarta Sans (wordmark uses 800 / ExtraBold).

Contents
--------
svg/
  unisync-mark.svg        Full-color mark, transparent. Scales to any size.
  unisync-mark-mono.svg   One-color teal silhouette (stamps, watermarks).
  unisync-icon.svg        App-icon tile (teal rounded square + cream mark).

icon/   App icons (teal tile, cream mark)
  1024 / 512              Square, no rounding — App Store / Play Store.
  256 / 192 / 180 / 120   Rounded — iOS & Android home screen.
  64 / 32                 Favicon / small UI.

mark/   Standalone glyph, transparent background
  unisync-mark-1024 / 512 / 256
  unisync-mark-mono-512

wordmark/
  unisync-wordmark-light  "UniSync" on transparent (dark text).
  unisync-wordmark-teal   "UniSync" on teal.
  unisync-lockup-light    Mark + wordmark, transparent.
  unisync-lockup-teal     Mark + wordmark, on teal.

unisync-loader.html       Standalone animated splash/loader (open in any browser).
                          The mark builds layer by layer, holds, and rebuilds.

Clear space: keep at least half the mark's height clear on every side.
Don't recolor outside the palette, rotate the mark, or add effects.
